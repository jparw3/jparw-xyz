# Next.js Starter Template

> A Next.js starter project template.

## Features

- ESLint + Prettier
- Tailwind CSS
- Framer Motion

### Installation

```bash
# clone repo
git clone git@github.com:jparw/next-starter.git my-project

# change into project directory
cd my-project

# install dependencies
npm install
```

### Development

```bash
# run development server
npm run dev
```
